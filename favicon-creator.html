Create Favicon Files

Your SVG favicon file has been created, but you still need to create the PNG and ICO files.

Option 1: Use an Online Favicon Generator

  1. Go to favicon.io or RealFaviconGenerator
  2. Upload the favicon.svg file you already have
  3. Download the generated package
  4. Replace the empty files in your project with the generated ones, making sure to use these filenames:
    • favicon-96x96.png
    • favicon.ico
    • apple-touch-icon.png

Option 2: Convert SVG to PNG and ICO using MacOS Preview and Online Converters

  1. Open the favicon.svg in a browser
  2. Take a screenshot (Cmd+Shift+4, then space and click on the browser window)
  3. Open the screenshot in Preview
  4. Resize to 96x96 pixels for favicon-96x96.png and 180x180 for apple-touch-icon.png
  5. Save as PNG
  6. Use an online converter like Convertio to convert the PNG to ICO format
